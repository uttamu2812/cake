# Cake Shop Website

## Overview
This project is a simple and responsive cake shop website that displays various cakes along with their details such as name, flavor, description, sizes, shapes, pricing, and images. The data is fetched from a JSON file, making it easy to manage and update the cake information.

## Project Structure
```
cake-shop-website
├── src
│   ├── index.html        # Main HTML document for the website
│   ├── css
│   │   └── styles.css    # Styles for the website
│   ├── js
│   │   └── app.js        # jQuery code to fetch and display cake data
│   └── data
│       └── cakes.json    # JSON file containing cake data
├── README.md             # Documentation for the project
```

## Features
- Responsive design that adapts to different screen sizes.
- Dynamic loading of cake data using jQuery.
- Attractive layout and styling for a pleasant user experience.

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the `src` directory.
3. Open `index.html` in a web browser to view the cake shop website.

## Running the Website
Simply open the `index.html` file in any modern web browser. The website will load the cake data from `cakes.json` and display it dynamically.
